# AI Text Summarizer

An AI-powered text summarization web app built with React and Tailwind CSS.

## 🚀 Features
- Paste any long text and get a concise AI-generated summary
- Clean and responsive UI with gradient design
- Real-time AI status indicator

## 🛠️ Built With
- React
- Tailwind CSS v3
- Vite
- Puter.js (AI API)

## 🏃 How to Run Locally
1. Clone the repository
   git clone https://github.com/FathimaFaziya-del/AI-TEXT-SUMMARIZER.git
2. Install dependencies
   npm install
3. Start the development server
   npm run dev
4. Open http://localhost:5173 in your browser
