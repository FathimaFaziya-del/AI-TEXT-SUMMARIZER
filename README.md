# 🤖 AI Text Summarizer

An AI-powered text summarization web app built with React and Tailwind CSS. Paste any long text and get a clean, concise summary in seconds.

---

## 🚀 Features

- 📝 Paste any long text and get an AI-generated summary instantly
- 📏 **Summary length selector** — choose Short, Medium, or Detailed
- 🕓 **Summary history** — revisit your past summaries in the same session
- 📊 **Live word count** — see word count and reading time before summarizing
- ⚡ Real-time AI status indicator
- 🎨 Clean responsive UI with dark gradient design
- 📱 Mobile friendly layout

## 🏃 How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/FathimaFaziya-del/AI-TEXT-SUMMARIZER.git
```

2. Go into the project folder
```bash
cd AI-TEXT-SUMMARIZER/my-project
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

5. Open your browser at
```
http://localhost:5173
```
